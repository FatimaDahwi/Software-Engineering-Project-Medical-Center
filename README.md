# Medical Center Management System

## Overview
A software engineering project developed to manage the operations of a medical center. The system helps patients, doctors, schedulers, and lab specialists manage appointments, medical records, prescriptions, billing, and laboratory services.

## Features
- Appointment booking and cancellation
- Medical history management
- Prescription requests and tracking
- Billing and payment management
- Doctor and patient management
- Laboratory test requests and results
- Appointment scheduling and notifications

## Functional Modules
### Patient
- Book appointments
- Cancel appointments
- Access medical history
- Request prescriptions
- Pay bills

### Doctor
- Access patient records
- Manage appointments
- Prescribe treatments
- Request laboratory tests

### Scheduler
- Schedule and manage appointments
- Allocate resources
- Send notifications and reminders

### Lab Specialist
- Manage laboratory test requests
- Track samples
- Report test results
